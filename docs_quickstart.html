Quick Start Guide

Welcome to Alconio! This guide will help you get up and running with our analytics and performance monitoring platform in under 5 minutes. We've designed the integration process to be as seamless as possible, whether you're building a modern web app or a complex enterprise system.

What you'll need:

  • An active Alconio account
  • Your Project API Key (found in Settings > Developer)
  • A website or application to monitor

Step 1: Install the SDK

The easiest way to integrate Alconio is via our official client libraries. We support all major package managers and environments.

terminal NPM / Yarn
npm install @alconio/sdk

Step 2: Initialize the Client

Once installed, import the client and pass your API key to authenticate your application.

import { Alconio } from '@alconio/sdk';

const client = new Alconio({
  apiKey: 'alc_live_xxxxxxxx',
  propertyId: 'site_1234'
});

Step 3: Start Tracking

Alconio automatically tracks page views and core web vitals, but you can also track custom events to understand user behavior better.

client.track('button_click', {
  button_id: 'signup_hero',
  color: 'blue'
});
