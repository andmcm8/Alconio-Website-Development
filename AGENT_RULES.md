# Agent Rules for This Project

The following rules apply to all AI agents working on this project:

1. **Initialize Git**: Ensure git is initialized in this project.
2. **Working Baseline**: Create a commit of the current state called 'working baseline' if it hasn't been done.
3. **Commit Before Changes**: From now on, before making any changes requested by the user, **always create a git commit first** so we can roll back if something breaks.
